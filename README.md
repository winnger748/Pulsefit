# ⚡️ PulseFit — Динамичный Landing Page для фитнес-индустрии

Этот проект — исследование того, как визуальная агрессия и энергия могут сочетаться с технической легкостью. PulseFit был разработан не просто как страница, а как инструмент для мгновенного вовлечения пользователя через скорость и динамику.

## 🛠 Мой подход и этапы разработки (Workflow)
Вместо использования готовых шаблонов, я разбил процесс на три итерации:
1. **Проектирование структуры:** Создание логической карты блоков, где основной акцент сделан на конверсию (Hero -> Motivation -> Pricing).
2. **Техническая сборка:** Написание чистого HTML/CSS кода. На этом этапе я использовал ИИ для рефакторинга сложных CSS-сеток, чтобы обеспечить идеальное выравнивание элементов.
3. **Оптимизация и анимация:** Интеграция JavaScript для "оживления" интерфейса без потери производительности.

## ⚙️ Технические решения
Чтобы проект "летал", я отказался от тяжелых библиотек в пользу нативных решений:
* **No-Framework Approach:** Использование чистого HTML5 и SCSS. Это позволило избежать лишнего кода (overhead), который часто встречается в Bootstrap или Tailwind.
* **Smart Animations:** Все эффекты появления реализованы через `Intersection Observer API`. Это значит, что анимация срабатывает только тогда, когда пользователь видит блок, что экономит ресурсы браузера.
* **Refactoring:** С помощью ИИ-инструментов был проведен аудит кода на предмет дублирования стилей, что сократило итоговый размер CSS-файла.

## 🎯 Ключевые показатели
* **Responsiveness:** Сайт прошел тестирование на устройствах от iPhone SE до мониторов 4K.
* **Performance:** Высокие показатели в PageSpeed Insights благодаря оптимизации активов и отсутствию внешних зависимостей.
* **UX:** Интуитивная навигация и формы, готовые к интеграции с любым бэкендом.
